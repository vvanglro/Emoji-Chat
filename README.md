# Emoji-Chat

Emoji-Chat is a simple chat application that allows users to send messages as emoji.


## Features
![chat.png](chat.png)

## Usage
1. Clone the repository:
    ```shell
    git@github.com:vvanglro/Emoji-Chat.git
    ```
2. Install dependencies:
    ```shell
    pdm install
    ```
3. Run the application:
    ```shell
    python main.py
    ```
